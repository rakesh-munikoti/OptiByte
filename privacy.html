Privacy Policy

Last Updated: June 3, 2026

1. Commitment to Privacy

At OptiByte, privacy is not an afterthought—it is the core design principle of our tool. We believe that your prompt data, documents, and corporate records are highly sensitive and should never be stored, harvested, or transmitted unnecessarily.

2. 100% Client-Side Prompt Optimization

All core prompt compression, tokenizer calculations, synoynm replacement mapping, and vocabulary adjustments run purely in your local browser-sandbox client memory. The text you type, paste, or compress never leaves your device and is never sent to our servers.

3. Document Upload & File Processing

When you use the document uploader (PDF, DOCX, XLSX, etc.), the file is transmitted securely (via HTTPS) to our lightweight conversion endpoint. The server extracts the text using the Python markitdown package, returns the extracted markdown to your browser, and immediately deletes the file from our storage. We do not keep logs of your document contents, nor do we train models or profile users using these files.

4. Analytics and Cookieless Tracking

We use Google Analytics 4 (GA4) to analyze anonymous website traffic volume, referral channels, and device types. This allows us to optimize the website experience. We do not track personal identifying information, IP addresses are anonymized, and we do not track prompt contents or user files inside the tracking platform.

5. Feedback Forms

When you submit a feedback form, the submitted rating and suggestion text are stored in our feedback logs purely anonymously. No email address, IP address, or identity handles are associated with feedback records unless you explicitly write them in your message text.

6. Security Measures

We employ production-grade security headers (Helmet), HTTPS-enforced TLS encryption, input sanitization libraries, and strict API rate limiting to safeguard the service from brute-force scans and malicious abuse.
